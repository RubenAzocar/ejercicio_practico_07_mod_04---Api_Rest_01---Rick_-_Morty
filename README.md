# Ejercicio Práctico - API REST Rick and Morty

## Descripción

Esta aplicación web permite consumir la API REST de Rick and Morty para mostrar información de los primeros 10 personajes. Incluye funcionalidades para listar personajes, agruparlos por especie y mostrar la ficha individual de cada uno. Optimiza el consumo de la API almacenando los datos en memoria tras la primera consulta.

## Funcionalidades

- **Obtener lista de personajes:** Muestra los 10 primeros personajes con sus atributos principales.
- **Agrupar por especie:** Presenta los personajes agrupados y ordenados por especie.
- **Ficha individual:** Permite ver la información detallada de un personaje por su ID (1-10).
- **Optimización:** Los datos se obtienen de la API solo la primera vez; luego se usan desde memoria local.

## Uso

1. Abre el archivo `index.html` en tu navegador.
2. Utiliza los botones para interactuar con la API y visualizar la información.
3. Ingresa un ID entre 1 y 10 para ver la ficha individual de un personaje.

## Tecnologías

- HTML, CSS, JavaScript
- API REST: [Rick and Morty API](https://rickandmortyapi.com/documentation)

## Ejemplo de salida

- Lista de personajes:
  - ID: 1 - Nombre: Rick Sanchez - Especie: Human
  - ...
- Agrupación por especie:
  - Human: Rick Sanchez, Morty Smith, ...
  - Alien: Abadango Cluster Princess
- Ficha individual:
  - ID, Nombre, Especie, Imagen
